CAD
===

CAD is a Creative Accounting Detection System support for:
- Choice item for analysis
- Analysis accounting
- Graphic
===
Requirement:
- PHP 5.2.9 (OR than)
- MYSQL 5
- APACHE 2.2 (Or than)
====
Install: 
- Copy source code to web Root directory
- Attach yii_tour.sql in directory
- Configure in protected/config/main.php
